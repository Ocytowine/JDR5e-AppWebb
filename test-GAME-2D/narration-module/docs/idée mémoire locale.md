
Mémoire locale

elle viendra compléter la mémoire globale (wiki) propre à la partie en cours, elle peut avoir une durée de vie pour éviter la surcharge de donnée par partie.

Elle ce créé quand un élément de jeu lié au lore revient dans la narration, elle peut aussi être créée par des éléments de jeu (ex: PNJ) ou par des actions du joueur (ex: entrer dans un batiment). cette mémoire locale peut être utilisée pour enrichir les descriptions, les dialogues ou les options de jeu, en se basant sur les éléments déjà rencontrés par le joueur.

la structure des éléments sauvegardés est suis les templates : wiki\Template
