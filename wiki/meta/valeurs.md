# Valeurs de base du jeu

## les unitées de temps

    1 année = 12 mois
    1 mois = 30 jours
    1 jours = 24 heures
    1 heure = 60 minutes
    1 minute = 60 secondes

## les unitées de distance

    ont utilise le mètre

## les dates

une date s'écrit : le numéros du jour du mois nommé, de l'année exemple : nous sommes le 17eme de Nara

les noms des 12 mois dans l'ordre :
    Naven
    Narima
    Nara
    Odekun
    Odorun
    Odrika
    Sumen
    Simari
    Sive
    Fanika
    Funara
    Feras

par saison :
    hiver : Natar -> Naven, Narima, Nara
    printemps : Odrinar -> Odekun, Odorun, Odrika
    été : Selar -> Sumen, Simari, Sive, 
    Autonome : Falar -> Fanika, Funara, Feras

## les saisons

Dunia possedent 4 saisons :
en langue commune :

    hiver : Natar
    printemps : Odrinar
    été : Selar
    Autonome : Falar

## Les monnaies

sur Dunia les monnaies courantes sont les pieces.
il existe 4 variantes de valeurs différentes :
    Les pieces de cuivre = valeur 1 (abréger PC)
    Les pieces d'argent = valeur 10 (abréger PA)
    Les pieces d'or = valeur 100 (abréger PO)
    Les pieces de platine = 1000 (abréger PP)
